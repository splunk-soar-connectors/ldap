The app will first try to connect to the server using ssl. If a secure connection cannot be made, then a non-ssl connection will be tried, if the config allows.

Many actions like get system attributes, change system ou etc. work on a host name. The app supports the following formats as input:

  • Host name format for e.g. WIN10ENTX64 (name in AD nomencalture)
  • DNS Host name for e.g. Win10Entx64.corp.contoso.com (dNSHostName)
  • Distinguished name format for e.g. CN=WIN10ENTX64,CN=Computers,DC=corp,DC=contoso,DC=com. (distinguishedName)
